ADVANCED #1

Write a piece of JavaScript code that when executed in the JavaScript console this page https://www.simoahava.com/analytics/element-visibility-trigger-google-tag-manager/, it finds all the outbound links in the content (SEE BELOW), and dynamically adds the attribute data-gtm-link="outbound" to these elements. "Outbound link" here means any link element whose href does not contain the string " simoahava.com ".

		<div class="entry-content">

Answer

		document.querySelectorAll('a').forEach(link => {
			if (link.host != "www.simoahava.com") {
				link.setAttribute("data-gtm-link", "outbound");
			}
		})
